# HomeLab

Personal homelab — self-hosted services running across two machines.

### Repository Structure

```
homelab/
├── rpi4/    # Raspberry Pi 4 — network services, VPN, dashboard
└── n100/    # N100 Mini PC — photos, backups
```

Each host directory is self-contained with its own `docker-compose.yml`, `.env`, and `README.md`.

→ [rpi4/README.md](rpi4/README.md)
→ [n100/README.md](n100/README.md)
